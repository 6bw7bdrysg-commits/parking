# 🚗 ParkKarma - Mobile Parking Community App

Το **ParkKarma** είναι μια προηγμένη full-stack mobile-first web εφαρμογή (Progressive Web App) που μετατρέπει την εύρεση στάθμευσης στην πόλη σε ομαδικό άθλημα. Οι χρήστες μπορούν να βλέπουν live ελεύθερες θέσεις, να δηλώνουν πότε αδειάζουν τη δική τους κερδίζοντας πόντους Karma, να κάνουν κράτηση μιας θέσης πριν φτάσουν, να φιλτράρουν με βάση ειδικές ανάγκες και πολλά άλλα.

---

## 🚀 Κύρια Χαρακτηριστικά (Features)

1. **🗺️ Live Χάρτης & Αναζήτηση (OpenStreetMap & Geocoding):**
   - Χρήση καθαρού χάρτη OpenStreetMap με μέγιστο επίπεδο ζουμ (Level 19).
   - Μπάρα αναζήτησης διεύθυνσης (Geocoder) για εύρεση θέσεων στον προορισμό σας.
   - Δυναμική εύρεση διεύθυνσης (Reverse Geocoding) που εμφανίζει την οδό και την πόλη μέσα σε κάθε πινέζα.

2. **🟣 Έξυπνο Σύστημα Κράτησης («Έρχομαι!»):**
   - Ένας οδηγός μπορεί να πατήσει «Έρχομαι!» σε μια θέση. Η πινέζα γίνεται **Μοβ 🟣** για 5 λεπτά, ενημερώνοντας τους άλλους να μην κατευθυνθούν εκεί.
   - Αν ο οδηγός δεν φτάσει στα 5 λεπτά, η κράτηση ακυρώνεται αυτόματα από το backend.

3. **📡 Ραντάρ Θέσεων (Radar Mode & Audio Alert):**
   - Η εφαρμογή ελέγχει συνεχώς αν δημοσιεύεται νέα ελεύθερη θέση σε ακτίνα κάτω του 1 χιλιομέτρου από την τρέχουσα τοποθεσία του χρήστη.
   - Μόλις βρεθεί, παράγεται ένας ηχητικός κώδικας ("Μπιπ!") και εμφανίζεται live ειδοποίηση (Toast Notification) στην οθόνη.

4. **🏷️ Γρήγορες Ετικέτες (Tags) & Φίλτρα:**
   - Δυνατότητα προσθήκης ετικετών κατά τη δημοσίευση: *🚙 Μεγάλο όχημα*, *♿ Θέση ΑμεΑ / Ράμπα*, *🌳 Σκιά*, *⚡ Φορτιστής EV*.
   - Αιωρούμενο πάνελ φίλτρων στο κάτω μέρος της οθόνης για άμεσο ξε
